# 「聴くtoノート」サポート

iOSアプリ「聴くtoノート」のサポートページです。

## よくある質問

### 録音が途中で止まっていました

画面ロックやほかのアプリの利用中も録音は続きます。電話に出たときやSiriを使ったときは
自動で一時停止し、終わると自動で続きから再開します。アプリを上スワイプで終了した場合や
電池切れの場合は録音が止まりますが、そこまでの録音は保存されています。
詳しくはアプリ内の「設定 → よくある質問」をご覧ください。

絶対に録り漏らしたくない会議では、機内モードにするのがいちばん確実です。
本アプリは録音も文字起こしも端末の中で完結するため、機内モードのままお使いいただけます。

### 文字起こしの精度を上げたい

- iPhoneを話し手の近く（30cm〜1mが目安）に置いてください
- 人名・社名・専門用語は「単語帳」に登録すると、その言葉で正しく書かれるようになります
- 「AI用語集」を使うと、お仕事やジャンルに合わせた用語集をAIがまとめて作れます

### AI議事録に必要なAPIキーとは？

AI議事録・Q&Aは、お手持ちのAIサービス（Gemini / Claude / ChatGPT）のAPIキーを設定して
使う機能です。Geminiのキーはクレジットカード不要・無料で取得できます。
取得手順はアプリ内の設定画面に案内があります。

### 初回のモデルダウンロードについて

文字起こしに使うAIモデル（約160〜230MB）を初回のみダウンロードします。Wi-Fi環境での
ダウンロードをおすすめします。以降はオフラインで動作します。
